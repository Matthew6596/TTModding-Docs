# TTModding-Docs

Welcome to **TTModding-Docs**, the home of TTGames LEGO video game modding — Everything you need, all in one place..

This repository contains the source of the **TTModding.com** website, built using **mdBook** and automatically generated and deployed via **GitHub Actions**.

Every change pushed to the main branch is automatically published as a **GitHub Pages** site and served live on **https://ttmodding.com**.

---

## 📚 What is this repository?

This repo hosts all the documentation related to:
- TTGames LEGO video game modding
- File formats, tools, research, and discoveries
- Guides, tutorials, and references for modders
- Community knowledge gathered over time

The goal is to centralize and preserve modding knowledge in a clean, readable, and community-driven format.

---

## 🤝 Community-driven project

This project is **open to everyone**.

You don’t need special permissions to contribute — if you have something useful to share, you’re more than welcome to do so.

You can contribute by:
- Adding new documentation pages
- Improving or correcting existing content
- Sharing research, findings, or reverse-engineering notes
- Fixing typos or clarifying explanations
- Adding links to tools, resources, or repositories

If it helps the community, it belongs here.

---

## 🛠️ How to contribute

Contributing is simple and entirely done through **Pull Requests**:

1. Fork this repository
2. Make your changes (add or edit Markdown files)
3. Open a Pull Request explaining what you added or changed

That’s it!  
Once your PR is reviewed and merged, your contribution will be **automatically deployed** and available on **TTModding.com**.

No build steps or deployment setup required on your side.

---

## ❤️ Credits

If you contribute, feel free to add yourself to the `src/misc/contributors.md` file with:

Every contribution matters, and credit is important.
